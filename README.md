# 🎓 Course Center Management System

A simple Course Center Management System built with **Python** using **Object-Oriented Programming (OOP)** principles.

This project simulates a training center where students can enroll in courses, search for available courses, and view course details. It also provides an admin mode for managing the system.

---

## 🚀 Features

### 👨‍💼 Admin Mode

* Secure admin login
* Display all students
* Display all courses
* Search for a course
* Delete a student
* Show statistics for each course

### 👨‍🎓 Guest Mode

* Add a new student
* Search for available courses
* dispaly courses 
* Enroll in one or more courses
